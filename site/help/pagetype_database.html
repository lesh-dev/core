Синтаксис config.dbs

Каждая строка задает одну колонку в таблице. Общий синтаксис (без фигурных скобок): {имя}:{тип}:{тип индекса}:{русское имя}:{тип ввода} Имя. Любое латинское имя без пробелов, которое будет обозначать данную запись.
Тип. Тип выражения. Бывает строковый, булевый, текстовый (несколько строк), картинка. Подробнее -- см подсказку в самом редакторе.
Тип индекса. Тип индекса. Бывает всего трех типов: primary, index и [none] -- следует указывать пустую строку. Рекомендовано primary колонку делать первой.
Русское имя Строка с пробелами (но без двоеточий), которая визуально замещает реальное имя.
Тип вводаРеализует систему выбора полей в "строковых" переменных. (Внимание! ТОЛЬКО в строковых!) Синтаксис: TAGLIST|{tagset} - Список полей taglist
FIELDS|{dbid}:{fieldname} - Список значений базы dbid/fieldname
STATIC|text - Статический текст Ввод может "суммироваться", например: TAGLIST|maxnum+FIELDS|base:priname В этом случае элементы "склеиваются" (без разделительного знака)
